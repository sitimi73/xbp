Arduinoとは

マイクロコントローラ（小さいコンピュータ）を搭載したマイコンボードと呼ばれる基盤の一種である。Arduino IDEというソフトウェアを用いてコードを転送し動作させる。

振り返り

ブレッドボードと言われる差し込むだけではんだ付けなしで電子回路を組める実験用の板を使い、LEDランプの点灯、フォトレジスタを利用した明るさで抵抗が変化するランプの点灯を行った。

やったこと

・LEDランプ点灯 ・抵抗器とフォトレジスタを利用した明るさで点灯のオンオフが変わる ・AIを利用してコードを書き五色のランプがランダムに点灯するようにした。

コード


#include <Adafruit_NeoPixel.h>
#define PIN        9 // 信号用のピンを指定
#define NUMPIXELS 5 // LEDの数を指定
int brightness=50;//明るさ

Adafruit_NeoPixel pixels(NUMPIXELS, PIN, NEO_GRB + NEO_KHZ800);

void setup() {
  pixels.begin(); // NeoPixel出力ピンの初期化
  pixels.setBrightness(brightness);
}

void loop() {
  pixels.clear(); // すべてのLEDの色を0にセット

  for(int i=0; i<NUMPIXELS; i++) {
    int r = 0;
    int g = 0;
    int b = 0;

    int mainColor = random(0, 3);

    if(mainColor == 0) {
      r = 255;
      g = random(0, 100);
      b = random(0, 100);
    } else if(mainColor == 1) {
      g = 255;
      r = random(0, 100);
      b = random(0, 100);
    } else {
      b = 255;
      r = random(0, 100);
      g = random(0, 100);
    }

    pixels.setPixelColor(i, pixels.Color(r, g, b));
    pixels.show();
    delay(50);
  }
}
